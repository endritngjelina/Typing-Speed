# ⌨️ Typing Speed

A pure vanilla **HTML, CSS, and JavaScript** typing speed test application —
no libraries, no frameworks, no dependencies. Developed iteratively across
8 versions (v0 → v7), this project measures your **WPM (words per minute)**
and **accuracy** in real time with a clean, minimal interface.

---

## ✨ Features

- ⏱️ Real-time WPM calculation as you type
- 🎯 Accuracy tracking (correct vs. incorrect keystrokes)
- ⏳ Countdown timer to control test duration
- ✅ Instant character-by-character feedback (correct/incorrect highlighting)
- 🔄 Restart/reset functionality for repeated tests
- 📱 Responsive layout — works on desktop and mobile
- 🚫 Zero external dependencies — pure vanilla JS
